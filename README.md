# 🎓 مسارات - نظام تتبع إنجازات الطلاب

**"جيل من الطموح والتميز"**

نظام إلكتروني لتتبع وتسجيل إنجازات الطلاب في مدرسة أبها الأهلية الابتدائية.

---

## 📋 نبذة عن النظام

مسارات هو نظام حديث لتتبع إنجازات الطلاب يعتمد على نظام النقاط والمستويات التحفيزية. يمكّن النظام المديرين من إدارة سجلات الطلاب، والمعلمين من التصويت للطلاب المتميزين، ويوفر لوحات شرفية شفافة لجميع المستخدمين.

### المميزات الرئيسية

- 🏆 **نظام النقاط** - تتبع ومكافأة تقدم الطلاب
- 📊 **المستويات التحفيزية** - خمس مستويات من "مُبادر" إلى "قُدوة"
- 🗳️ **تصويت المعلمين الأسبوعي** - مشاركة ديمقراطية في تكريم الطلاب
- 📈 **لوحة الشرف** - عرض أفضل الطلاب
- 👥 **صلاحيات متعددة** - واجهات منفصلة للمديرين والمعلمين والزوار
- 📱 **تصميم متجاوب** - متوافق مع الجوال والتابلت والكمبيوتر

## 📦 التثبيت والتشغيل

### المتطلبات

- Node.js 22 أو أحدث
- pnpm 9 أو أحدث
- MySQL 8 أو متوافق

### خطوات التشغيل

1. **استنساخ المشروع**
   ```bash
   git clone https://github.com/w2ksa/masarat-edu-tracker.git
   cd masarat-edu-tracker
   ```

2. **تثبيت المكتبات**
   ```bash
   pnpm install
   ```

3. **إعداد ملف البيئة**
   ```bash
   cp .env.example .env
   ```
   
   عدّل ملف `.env` وأضف:
   - `VITE_ADMIN_CODE` - كود المدير
   - `VITE_TEACHER_CODE` - كود المعلمين
   - `DATABASE_URL` - رابط قاعدة البيانات
   - `JWT_SECRET` - مفتاح التشفير

4. **إنشاء قاعدة البيانات**
   ```bash
   pnpm db:push
   ```

5. **تشغيل المشروع**
   ```bash
   pnpm dev
   ```

6. **افتح المتصفح على**
   ```
   http://localhost:3000
   ```

## 🎯 المستخدمون

### 1. **المدير**
- إضافة/تعديل نقاط الطلاب
- إدارة المعلمين
- فتح/إغلاق فترات التصويت
- عرض التقارير والإحصائيات

### 2. **المعلمون**
- التصويت لـ 3 طلاب متميزين أسبوعياً
- كل تصويت يضيف 10 نقاط للطالب
- عرض الطلاب حسب الصف

### 3. **الزوار**
- تصفح جميع الطلاب ومستوياتهم
- البحث عن الطلاب بالاسم
- عرض أفضل 5 طلاب
